# **App Name**: AquaFolio

## Core Features:

- Animated Hero Section: Implement a Hero section with animated introduction and interactive call-to-action buttons (Download CV, Contact Me).
- Animated About Me: Design an animated 'About Me' section that highlights skills using animated progress bars or icons.
- Experience Timeline: Create an animated timeline to display professional experiences and academic projects.
- Interactive Portfolio Grid: Develop an animated portfolio grid with hover effects to showcase web apps, mobile apps, and AI projects, complete with links to GitHub/live demos.
- Animated Contact Form: Implement a contact section featuring a stylish, animated form and social media links with hover animations.
- Theme Toggle: Integrate a dark/light theme toggle for a modern, user-friendly experience.
- Personalized Greetings: The app uses a tool: Generate personalized greetings based on the user's browsing history, if available, otherwise present a default greeting.

## Style Guidelines:

- Primary color: Subtle light blue (#A7D1F2) to invoke calmness and trust, representing the developer's reliable nature.
- Background color: Light grey (#F0F0F0) for a clean, minimal look, ensuring content is easily readable and not distracting.
- Accent color: A more saturated blue (#4B86B4) to highlight key interactive elements like buttons and links, providing clear visual cues.
- Body text: 'PT Sans', a humanist sans-serif suitable for both headlines and body text.
- Code Font: 'Source Code Pro' monospaced font.
- Use flat design icons with a line art style for skills, project categories, and social media links to maintain a clean and modern look.
- Incorporate subtle animations like fade-ins, slide-ins, and zoom effects on scroll to create a dynamic and engaging user experience. Use Framer Motion for smooth transitions and hover effects on project cards.
